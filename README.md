# Norsk Hangman

Hangman med 147 000 norske ord fra Norsk Ordbank (Bokmål).

No dependencies. No venv. No requirements.txt. Just Python 3.

## Spill

```
python3 hangman.py
```

Velg et ord fra lista, så kan noen andre gjette. Ordet vises i bunnen av skjermen så du ikke glemmer det.

Options:
- `--min-length 8` — lengre ord (default: 6)
- `--length 5` — eksakt lengde
